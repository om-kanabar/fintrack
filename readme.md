# FinTrack

---

### By: Om Kanabar

This is FinTrack, an expese tracker that will offer monthly insights into your expenses that are safely stored using Google Firebase. Authentication is seemless with Firebase Auth, and signin with google. 

I made this project because my mom wanted to have a journal with her for tracking her expenses, especially cash, so I decided to make a digital journal because physical journals get lost and can break, wheras digital tracking doesn't!
